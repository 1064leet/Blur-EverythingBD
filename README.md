# BlurEverything

BetterDiscord plugin that blurs images, GIFs, and videos until hovered.

## Installation
1. Requires [BetterDiscord](https://betterdiscord.app/) to be installed.
2. Download `BlurEverything.plugin.js`.
3. Move the file into your BetterDiscord plugins folder.
